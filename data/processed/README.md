# Bundled Processed Inputs

This folder contains only lightweight public inputs used by the reproducibility notebook.

- `agreement_sos_publishable_points_dedup.csv`
  Publishable point table for reproducing the agreement figure and summary metrics in manuscript Section 3.1.

- `agreement_sos_publishable_metrics_dedup.csv`
  Archived metric table associated with the public agreement dataset.

- `regional_temperature_sos_table.csv`  
  Lightweight regional year table used for the temperature-versus-SOS analysis in manuscript Section 3.5. 
  The temperature variables included in this table are derived from ERA5 reanalysis data, accessed through the Copernicus Climate Data Store.

These are derived analysis inputs, not raw Sentinel-2 time series and not raw in situ phenology observations.
